###### 10/30 강의
스택stack
- 데이터를 일시적으로 저장하기 위해 사용하는 자료구조 
- 데이터의 입력과 출력 순서는 후입선출
즉, 가장 나중에 넣은 데이터를 가장 먼저 꺼냄
- 푸시(push) : 스택에 데이터를 넣는 작업
- 팝 : 스택에서 데이터를 꺼내는 작업
- 꼭대기(top) : 푸시, 팝을 하는 위치
테이블에 겹겹이 쌓은 접시처럼 데이터를 넣는 작업도 꺼내는 작업도 위쪽부터 수행 
- 바닥(bottom) : 스택의 가장 밑바닥 부분
